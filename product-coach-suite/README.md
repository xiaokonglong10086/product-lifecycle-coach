# Product Coach Suite

用户实际安装一个 Skill：`product-coach-router`，中文显示名“产品教练”。

内部包含六个专业子模块和四个共享专业模块。总控采用可检查路由，不宣称确定性程序调用。

项目事实库：`PRODUCT_TRUTH.md`、`DESIGN_PRINCIPLES.md`、`DECISIONS.md`、`OPEN_QUESTIONS.md`、`CURRENT_DELIVERABLES.md`、`PM_LEARNING_LOG.md`。

功能验证是核心阶段：产品行为真实，底层服务可模拟，主功能 L3，关键跨端 L4，并提供产品实验室。
